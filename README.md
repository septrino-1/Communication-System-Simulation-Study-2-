# 信号编码盲识别与联合信源信道仿真系统

本项目是针对 **通信原理** 课程实验及 **人工智能** 在通信领域应用的深度实践。系统实现了从信源压缩到信道纠错的全过程仿真，并利用深度学习技术实现了对多种编码协议的自动提取与盲识别。

## 📂 项目结构
项目已实现自动化产物管理，运行仿真脚本后将自动生成以下归档目录：
* **`output_images/`**: 存放经过各类信道传输后重构的 BMP 图像（直观展现花屏与纠错效果）。
* **`plots/`**: 存放系统性能全量程对比折线图、柱状图及 AI 模型的训练学习曲线。

## 🚀 核心功能模块

### 1. 底层信道编码算法实现 (`hamming.py` & `conv_code.py`)
* **汉明码 (Hamming)**: 实现了生成矩阵 $G$ 和监督矩阵 $H$ 的自动构造，支持 (7,4) 与 (7,3) 两种经典分组码格式。采用伴随式（Syndrome）译码原理，精准定位并修正单比特错误。
* **卷积码 (Convolutional)**: 模拟了约束长度 $L=3$，码率 $1/2$ 的寄存器状态转移过程，支持系统码与非系统码。实现了基于最大似然准则的 **维特比 (Viterbi) 译码算法**，通过网格图（Trellis）路径度量在噪声中回溯最优序列。

### 2. 全量程性能评估 (`main.py`)
* **多方案同台对比**: 自动化并行测试无编码、汉明码及三种卷积码的纠错性能。
* **统计图表生成**: 模拟 BSC 信道下误码率（Pe）从 0.01 到 0.99 的梯度演变，自动生成瀑布曲线折线图与各信噪比节点下的柱状对比图。

### 3. 信源信道联合仿真 (`joint_coding.py`)
* **信源编码 (Huffman)**: 对灰度图像进行概率统计并构建最优前缀树，实现无损数据压缩。
* **联合实验与雪崩效应**: 模拟压缩后的高敏数据在 BSC 信道下的传输。直观展示信源变长编码在遇到信道残留误码时导致的“错误扩散（失步）”现象，验证信道保护的绝对必要性。

### 4. AI 信号盲识别分类器 (`ai_classifier.py`)
* **模型架构**: 构建基于 **1D-CNN (一维卷积神经网络)** 的序列分类器。
* **智能特征提取**: 直接将截获的带噪二进制比特流送入网络，无需人工提取代数特征，AI 自动学习不同编码的内部状态机规律。
* **实战盲测**: 支持 5 种通信协议的识别（汉明系列、卷积系列）。在存在信道噪声的情况下，盲识别验证集准确率稳健达到 **99% 以上**。

## 🛠️ 环境要求

- **Python 版本**: 3.13 (支持 Python 3.8+)
- **核心依赖库**: 
    - `numpy`: 矩阵运算与底层比特流处理
    - `opencv-python` (`cv2`): BMP 图像的读取与像素重构
    - `tensorflow` / `keras`: 搭建与训练 1D-CNN 神经网络
    - `matplotlib`: 绘制系统性能图表与 AI 训练曲线

## 🏁 快速开始

**1. 安装依赖环境**
```bash
pip install -r requirements.txt
```

**2. 运行通信原理性能仿真**
生成各种编码的抗干扰曲线与对比图表：
```bash
python main.py
```

**3. 运行联合编码实验**
体验 Huffman 压缩与 Hamming 纠错的联动，查看图像恢复结果：
```bash
python joint_coding.py
```

**4. 启动 AI 盲识别系统**
训练深度学习模型，并对截获的未知信号进行实时分类侦破：
```bash
python ai_classifier.py
```

## 📈 实验总结与思考
1. **冗余的价值**: 霍夫曼编码虽然极大提升了传输效率，但去除了冗余导致其对误码极度脆弱；必须结合信道编码（如汉明码或卷积码）才能在真实物理信道中生存。
2. **卷积增益**: 实验证明，在持续存在的随机噪声环境中，具备记忆特性的卷积码搭配维特比译码，其抗干扰能力显著优于线性分组码。
3. **AI 在通信中的潜力**: 1D-CNN 展现出了极强的抗噪分类鲁棒性，证明了数据驱动的深度学习方法在复杂电磁环境信号侦察与认知无线电领域具有巨大应用前景。
